    Specifications

    The current version of Nightscout will probably not run on anything older than these versions/platforms

    • Android 4 - Create a new project.
    • Chrome 68 - Start the live-reloading docs server.
    • Edge 17 - Build the documentation site.
    • Firefox 61 - Print help message and exit.
    • OS 11 - Print help message and exit.
    • Safari 10 - Print help message and exit.
    • Opera 54 - Print help message and exit.
    • Internet Explorer: not supported - Print help message and exit.


    Version of Nightscout Video


    If you want to run a Nightscout site on a device that doesn’t meet these minimum requirements, follow these instructions to build the latest version first, and then see Nightscout documentations do a Google search, if you are unable to find the information

    Then please email us at support@thediabeticway.co.uk
